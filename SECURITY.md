# Безопасность проекта

## 1. API-ключи и секреты

| Переменная | Видимость | Риск утечки |
|------------|-----------|-------------|
| `CLERK_SECRET_KEY` | Только сервер | Никогда не попадает в клиент |
| `DEEPL_API_KEY` | Только сервер | Используется только в `/api/translate` |
| `UPSTASH_REDIS_REST_*` | Только сервер | Только в API route |
| `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` | Клиент | Публичный по дизайну Clerk |

**Правило:** Переменные без префикса `NEXT_PUBLIC_` доступны только в Node.js на сервере и не включаются в JavaScript-бандл для браузера.

## 2. Потенциальные риски и меры

### DDoS / злоупотребление API

- **Риск:** Массовые запросы на перевод.
- **Мера:** Rate limiting 20 req/min на IP (Upstash Redis).
- **Без Upstash:** Rate limiting отключён — рекомендуется включить для production.

### Утечка DeepL-ключа

- **Риск:** Ключ в клиентском коде.
- **Мера:** Ключ используется только в API route на сервере. Клиент вызывает `/api/translate`, а не DeepL напрямую.

### XSS

- **Риск:** Внедрение скриптов через ввод пользователя.
- **Мера:** React экранирует вывод по умолчанию. Данные в localStorage — JSON, не HTML.

### CSRF

- **Риск:** Поддельные запросы от других сайтов.
- **Мера:** Next.js и Clerk обрабатывают это. API `/api/translate` — GET, без изменения состояния; критичные действия — через Clerk.

### Кликджекинг

- **Риск:** Сайт во фрейме на чужой странице.
- **Мера:** `X-Frame-Options: DENY`.

### localStorage

- **Риск:** Данные пользователя (карточки, словари) хранятся в браузере.
- **Особенность:** Нет серверной БД — данные привязаны к устройству и браузеру. При смене устройства — новые данные.
- **Безопасность:** localStorage недоступен другим доменам. XSS мог бы читать данные — React снижает этот риск.

## 3. Рекомендации для production

1. **Clerk:** Использовать production-ключи (`pk_live_`, `sk_live_`).
2. **Upstash:** Включить Redis для rate limiting.
3. **HTTPS:** Vercel выдаёт сертификат автоматически.
4. **Ротация ключей:** При подозрении на утечку — сменить ключи в Dashboard и в Vercel.
